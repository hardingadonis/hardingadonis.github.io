# Minh Vương | Portfolio

> My portfolio

## Requirements

- NodeJS: 22.20.0
- Package manager: pnpm

## How to develop

- Install dependencies:

  ```bash
  pnpm install --frozen-lockfile
  ```

- Run this web:

  ```bash
  pnpm dev
  ```

- Open brower and access [http://localhost:5173/](http://localhost:5173/).
